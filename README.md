# Catálogo Laravel (Productos)

CRUD de productos en Laravel usando:
- Rutas + Controladores
- Migraciones + Eloquent ORM
- Vistas Blade

## Requisitos
- PHP 8.1+
- Composer
- MySQL

## Instalación
```bash
git clone <https://github.com/jsbrad23/catalogo-laravel.git>
cd catalogo
composer install
cp .env.example .env
php artisan key:generate
